Autobuilding¶

You can set up auto-building on every save of a .rst file:

  1. Activate virtualenv, if not already active:

    source bin/activate

  2. Install watchdog tools, if not already installed:

    pip install watchdog

  3. Run watchmedo to monitor all .rst files:

watchmedo shell-command
–patterns=”.rst” –ignore-pattern=’_build/’ –recursive –wait –command=’make html’
